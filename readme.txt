== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Click Upload Theme, then choose the theme and install 
3. Click on the 'Activate' button to use your new theme right away.
4. Read the document for a guide on how to customize this theme.
5. Navigate to MeloKids in your admin panel and customize to taste.
